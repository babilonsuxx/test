Фронт сделан в виде класса на JS, что позволяет выводить несколько независимых таблиц
на одном экране с разными настройками.
Данные с фронта для разнообразия взял AJAX-ом.


Бэк php+mysqli 
Дамп базы прикреплен.

Функционал:
Сортировка по возрастанию и убыванию при клике на заголовок и фильтра, описанные в ТЗ

Украшений сделал по минимуму.
